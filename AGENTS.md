# AGENTS.md — Aurelion Dashboard Rules

## Project Overview
Aurelion is a productivity dashboard with focus timer, tasks, and stats.

## UI Design Rules
- Theme: Dark + yellow 
- Style: Soft glow, gradients, rounded corners
- Avoid harsh contrast
- Use glassmorphism where possible
- Icons: minimal + nature (amber, yellow, calm)

## Components
- Timer (circular progress)
- Task list with checkboxes
- Stats cards
- Progress ring

## Coding Rules
- Use React + Tailwind
- Components must be reusable
- Keep files under 300 lines
- Use functional components only

## UX Rules
- Smooth animations (no abrupt changes)
- Feedback on every click
- Focus on calm + minimal cognitive load

## When generating UI
- Always improve spacing
- Improve hierarchy
- Suggest better interaction patterns
